# Stellar Astrophysics Simulator: Blackbody Radiation & Binary Light Curves

A Python-based computational astrophysics tool designed to model stellar radiation environments using quantum mechanics principles, and simulate the photometry of eclipsing binary star systems.

## 🔭 Features
1. **Stellar Spectrum Generation:** Models the empirical spectral radiance of major stellar classes (Rigel, The Sun, Betelgeuse) using **Planck's Law**.
2. **Wien's Displacement Verification:** Automatically calculates and visualizes peak emission wavelengths ($\lambda_{\max}$) across different stellar temperatures.
3. **Binary Star Light Curve Simulation:** Simulates time-series photometric flux drops during primary and secondary eclipses in a binary system.

## 📊 Methodology & Physics Equations
* **Planck's Law:** $$B_\lambda(\lambda, T) = \frac{2hc^2}{\lambda^5 \left(e^{\frac{hc}{\lambda k_B T}} - 1\right)}$$
* **Wien's Law:** $$\lambda_{\max} T = b$$
